# Minicurso de Git e Github:

## Fluxo de trabalho local

1.  Definir o diretório que será utilizado no para o projeto
2.  git init . dentro do diretório
3.  git config
4.  git status
5.  git add nome_arquivo
6.  git status
7.  git commit -m "Mensagem Explicativa de Commit"
8.  git log
10. git switch -c nova_branch
11. Ao alterar e salvar o arquivo
12. Repetir os passos 5 a 8
13. git switch main
14. git merge nova_branch

## Fluxo de trabalho remoto -> local
