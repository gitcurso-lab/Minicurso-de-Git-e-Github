# Minicurso de Git e Github:

## Fluxo de trabalho local

1.  Definir o diretório que será utilizado no para o projeto
2.  git init . dentro do diretório
3.  git config
4.  git status
5.  git add nome_arquivo
6.  git status
7.  git commit -m "Mensagem Explicativa de Commit"
8.  git log
10. git switch -c nova_branch
11. Ao alterar e salvar o arquivo
12. Repetir os passos 5 a 8
13. git switch main
14. git merge nova_branch

## Fluxo de trabalho do local para o remoto

1.  Realizar o fluxo de trabalho local
2.  Criar repositório no Github
3.  git remote add origin <link_repositorio_github>
4.  git branch -M main
5.  git push -u origin main

## Fluxo de trabalho do remoto para o local

1.  git pull <link_repositorio_github>
2.  git clone <link_repositorio_github>
3.  git switch -c nova_branch
4.  alterar e salvar o arquivo
5.  git status
6.  git add nome_arquivo
7.  git status
8.  git commit -m "Mensagem Explicativa de Commit"
9.  git push origin nova_branch
10. abrir Pull Request para a main
